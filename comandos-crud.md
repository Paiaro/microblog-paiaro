# Operações CRUD

## RESUMO


- C: CREATE (INSERT) -> inserir dados
- R: READ (SELECT) -> ler/carregar dados
- U: UPDATE (UPDATE) -> atualizar
- D: DELETE (DELETE) -> excluir dados

## Exemplos
